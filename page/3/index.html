2021年度总结

发表于2022-01-31更新于2022-12-11

前言

2021这一年在今后的日子想必会是我人生当中浓墨重彩的一方篇章，作为第一次搭建博客网站的一年，在2022除夕夜热闹时刻我在git bash上打下了$hexo new的代码开始了农历新年最后的写作——年度总结，彻底的为过去的一年画上了一个句号。愿这一年的经历能够在时光的厚重中沉淀发酵，在人生走到一个进程时，回忆往昔，曾经的野望做到了多少，又会有多少的遗憾呢；那般思索的滋味肯定如同一壶陈年老酒的芳香吧。

高中阶段（1-6月）

今年的暑假将我的一段人生又分成了两个阶段吧，我从高中进入了大学的生活，那么我想这一年的总结也从此处流年吧。
其实准备高考的这个阶段从去年高中最后的一个寒假算起，是十分枯燥乏味的，若要落笔成章，也只能说说感悟了吧。“每个优秀的人，都有一段沉默的时光。那段时光，是付出了很多努力，却得不到结果的日子，我们把它叫做扎根。”这是习主席的一番话语，只是说来惭愧吧，我是在小黑盒第一次看到这个句子，我也总是用这样的话语来安慰自己，当时的寒假我正经历了二诊的失败（是真的考的好差的那种），满脑子都是担忧高考，甚至那段时间里一直都在考虑复读；从来没有想过自己竟然可以那么焦虑吧，不过那个时候的我还是个游戏少年，忧虑时也总能将自己寄托在那般虚拟的世界中得到一丝心灵的慰藉吧。
从寒假路过到达学校的时候，到底还是现实，到底还看成绩吧；最后冲刺的自己真的感觉沉浸在那般题海当中（当然了，周末依旧是控制不住自己上网玩游戏的啦），我记得最后的时光里我基本成为了完全不听课的刷题机器吧；每次一道题，一套卷的正确牵动着我的神经，给予我自信；感觉刷题给予的正反馈是支持我到高考的一个重要原因。
苦心人，终不负。终于让我熬到了高考，我记得刚入高考考场的时候，真的有命运皆系于笔下的紧张感，前半个小时的语文试卷我甚至就写完了论述文；不过我自认为我心态还是不错的，就这样吧，两天过了很长，却又觉得很短；是的，我毕业了，从绵中实验走向了更大的舞台。
至于高考之后的时间我在干嘛，书丢了，笔扔了，回到家里，电脑打开，直接开摆。

暑假

高考完了，当然就是暑假，要说这个时间段里，我还记得当时我拿到我高考成绩的时候我爸妈在我旁边，简直一脸嫌弃，以至于后面填高考自愿的时候也是出了不少幺蛾子（我爸妈坚持我应该选调公务员，走他们的路）。后面的事情呢我在七月份的一篇文章里写到了，也就偷懒了嘿嘿。

大一上

我的大学生活，和过去的我想象的有点不同，但是自己呢努力地向着心中的生活前行；在我看来，大学更像在体验过去不敢尝试的生活。在大学里终于有了时间去做那些曾经一直想做的却因为种种原因而瞻前顾后的事情。
其实我自己最大的改变就是有了更多的勇气去做取舍（取舍真的好难），我基本放弃了自己的保研资格去学习那些我感兴趣的东西：去打打竞赛，写写博客，做做项目。尽管自己的绩点学积分并不是那么优秀，但我却觉得十分值得。我一直相信最后真正的成功者是敢于去剑走偏锋的，听从本心吧，毕竟年轻是我们的资本，还可以试错呀，我不希望未来的自己总是活在当初的后悔中。
当然了回忆这半年来自己给自己的承诺还是完成了大半了，打进ACM选拔的复赛，打进蓝桥杯的省赛，多学一门编程语言，多会使用一个开发工具，搭建个人网站等等。但其实呢，原本自己想写写游戏资讯的，结果就帮别人改了稿子，自己基本没有码字过（主要因为是电子阳痿了，不想玩游戏，全年时常才200多小时），这还是遗憾呀。

写在最后

在除夕夜码字，属实是一番风味了，我new这篇文章的时候不过在23：01：00，结束时却又是到了凌晨一点了。我在乡下二楼的小房间里透过窗户还是可以看到外面是很热闹的，还有着小孩在放着鞭炮，就像当初我一样，12点的跨年烟火依旧准时，客厅长辈放春晚的声音也才刚刚停下，此时虽然一人在房间里码字，外面的热闹依旧属于每个人呀。
最后在回忆下今夜吧，从七点半开始我和我的hxd挤在一起看哔哩哔哩拜年祭（主要是为了那个抽奖嘿嘿）在八点的时候又参加了牛客网的春节acm比赛（原本挺有心情的，结果第二题没写long long一直wa就没写了）随后又继续了b站一直到了写文章构思的时候。
新年快乐，朋友们。

1.10题目原题链接：

acwing的每日一题从1.2开始，今天做的就是从1.2开始
奶牛贝茜正在学习如何在不同进制之间转换数字。
但是她总是犯错误，因为她无法轻易的用两个前蹄握住笔。
每当贝茜将数字转换为一个新的进制并写下结果时，她总是将其中的某一位数字写错。
例如，如果她将数字 14 转换为二进制数，那么正确的结果应为 1110，但她可能会写下 0110 或 1111。
贝茜不会额外添加或删除数字，但是可能会由于写错数字的原因，写下包含前导 0 的数字。
给定贝茜将数字 N 转换为二进制数字以及三进制数字的结果，请确定 N 的正确初始值（十进制表示）。

输入格式

第一行包含 N 的二进制表示，其中一位是错误的。
第二行包含 N 的三进制表示，其中一位是错误的。

输出格式

输出正确的 N 的值。

题解

此题可以用枚举出二进制中所有位数的错误情况，进而转化为三进制数，最后用一个变量判断和三进制数对应错误的个数，为一个则为所求值，然后输出就好。

1.11题目链接

贝茜对她最近在农场周围造成的一切恶作剧感到抱歉，她同意帮助农夫约翰把一批新到的干草捆堆起来。
开始时，共有 N 个空干草堆，编号 1∼N。
约翰给贝茜下达了 K 个指令，每条指令的格式为 A B，这意味着贝茜要在 A..B 范围内的每个干草堆的顶部添加一个新的干草捆。
例如，如果贝茜收到指令 10 13，则她应在干草堆 10,11,12,13 中各添加一个干草捆。
在贝茜完成了所有指令后，约翰想知道 N 个干草堆的中值高度——也就是说，如果干草堆按照高度从小到大排列，位于中间的干草堆的高度。
方便起见，N 一定是奇数，所以中间堆是唯一的。
请帮助贝茜确定约翰问题的答案。

输入格式

第一行包含 N 和 K。
接下来 K 行，每行包含两个整数 A,B，用来描述一个指令。

输出格式

输出完成所有指令后，N 个干草堆的中值高度。

题解

依然简单题，利用数组存放下所有值（都为0），得出原数组的差分数组，并利用差分数组进行题意的操作，最后利用sort函数排序取中间值即可

1.15周赛：数字

给定一个大于 2 的十进制正整数 A。
该数字在 2∼A−1 进制表示下的各位数字之和均可以求出。
例如，数字 123 在 16 进制表示下，共有 2 位：第 1 位是 7，第二位是 11，各位数字之和为 18。
现在，请你将 A 在 2∼A−1 进制表示下的各位数字之和全部相加，并将得到的结果除以 A−2，最终结果以最简分数形式输出。

输入格式

一个十进制正整数 A。

输出格式

输出格式为 X/Y，其中 X 表示输出答案的分子，Y 表示输出答案的分母

数据范围

前三个测试点满足 3≤A≤10。
所有测试点满足 3≤A≤1000 。

题解

虽然题标为中等题，但这道题就是以简单枚举得来的简单题。通过输入的数字可以枚举其2~A-1的和，然后利用约分的算法进行求答案输出即可。

12.12ACM选拔赛

这次比赛可以说真的没什么准备了，我也是在一次意外下看到了这个比赛信息，在这里还感谢飞鱼学长发的了。在我了解到了有这次比赛的时候，其实只有一周时间了，不过呢我也挺庆幸自己先前为了蓝桥杯所准备的东西，让这次的比赛还是有一定的准备。当然时间有限的情况下自己也只能是每天在leetcode上进行ACM有关刷题然后就草草了事。
考试当场可以说ACM比赛的紧张感是十足的，特别是自己忘了转义字符的知识点害得第一道签到题直接不会，前一个小时的时间甚至只拿下了第二题的AC，全程就是很紧张的答题，只要有一个气球的获得就是一个AC，我也很紧张看看自己的排名，封榜前甚至只有70多名。不过最后一个小时两题的AC还是让我进入了前30，获得了ACM下轮选拔的资格，也算是达成了自己的一个目标了
赛后我自己也认真反省了自己，我和真正的oler还是有不小的差距的，而下周就是蓝桥杯的比赛，所以在最后一周的时间又刷了一定的题量，学了一些算法，希望能够缩小自己的差距。

12.19蓝桥杯选拔

经过了上周的比赛后，这次的比赛有个很大的进步，那就是心态问题了，全场都是以一种很平静的心情去做题，甚至还提前了一个小时交卷开溜（主要是感觉题比起ACM简单了不少），赛后也是进行了总结，自己对于数据结构基本还是没啥掌握，以至于填空的第四题连题意都没看懂，当然最为遗憾的还是第一题进制转化出错。不过呢编程大题的稳定发挥还是让我进入了前100获得了省赛推荐，也是没了遗憾，只求下次加油啦。

总结

刷题刷题刷题，太重要了。
